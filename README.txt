D {0-1} KP 背包问题求解器
基于动态规划的 D {0-1} KP（折扣 0-1 背包问题）求解工具，支持数据读取、可视化绘图、排序、最优解求解与结果导出。

一、项目简介
D {0-1} KP 是经典 0-1 背包问题的扩展模型，每组包含 3 个物品，每组最多选择一个。本项目使用动态规划算法实现最优求解，并提供图形界面，方便操作与测试。

二、功能清单
✅ 读取 D {0-1} KP 标准数据集（.txt）
✅ 绘制重量 - 价值散点图
✅ 按项集第三项价值 / 重量比降序排序
✅ 动态规划求解最优价值与选择方案
✅ 结果保存为 TXT 文件
✅ 结果导出为 Excel 文件
✅ 友好图形界面（GUI）
✅ 自动过滤注释行、空行，兼容多种数据格式

七、算法说明
1、算法：动态规划（DP）
2、复杂度：O (N × C)
3、约束：每组最多选一个物品
4、目标：不超过背包容量时价值最大化
